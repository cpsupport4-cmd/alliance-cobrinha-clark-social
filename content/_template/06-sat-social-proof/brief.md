# SAT · Social Proof

**Pillar:** Community Content (scale signal)
**Drive source:** `Events / Group Photos`
**Format preference:** Group photo · packed class · open mat shot · event poster
**Goal:** FOMO — show the room is full

## Caption tone

Quietly confident. Don't oversell. The image does the work.

- "Weekend training with the team."
- "Saturday open mat. Show up. Roll. Stay for coffee."
- "42 of us, building something."

**Avoid:** "Limited slots", "Don't miss out", anything that sounds like a flash sale.

## Brief override

If this Saturday is a community event week (Outdoor Activity / Kids+Parents Day / Workshop / Roll+Coffee), drop the event details here. The skill will pull the matching event poster template instead of a default group photo layout.
